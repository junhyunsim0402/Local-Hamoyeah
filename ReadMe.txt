#하모예(HAMOYEAH)


지역의 문화재, 착한식당, 제로 웨이스트 샵 등등을 찾아가 인증하면 포인트를 주며 시민들의 참여를 통해 지역 **경제가 살아나는 선순환 구조** 및** 공공데이터를 활용한 정주여건 시각화**를 통해 정보 제공한다.


###[개발 목적]
- - -
1. **지역 경제 선순환 구조 구축**: 지역 문화재, 축제, 가맹점 방문 인증 시 포인트를 지급하여 지역 화폐 사용을 장려하고, 이를 통해 소상공인 매출 증대와 시민 혜택이 공존하는 생태계 조성한다.
2. **데이터 기반 '정주여건' 제공**: 4대 핵심 공공데이터(CCTV, 가로등, 대기질, 소음)를 융합·분석하여 지역별 점수와 등급을 산출, 시민들에게 직관적인 생활 환경 정보를 시각화하여 서빙한다.
3. **자발적 참여 중심의 시민 참여형 구조**: 보상과 정보 제공을 결합하여 시민이 능동적으로 지역 서비스를 이용하고 피드백하는 '데이터 기반 참여형 플랫폼' 완성한다.

- - -
장민서(팀장)
공공데이터 DB적재
프론트 엔드 개발 및 백엔드 연동
JIRA 관리
유튜브 api

이연지(발표자)
회원가입 시스템
즐겨찾기 시스템
관리자 및 인증 시스템
로그인 및 회원가입 연동

심준현(GIT 담당)
카카오맵 정주여건 시각화
카카오맵 마커 기능
위도/경도 거리 계산
뉴스 크롤링


###[기술 스택]
- - -
+ Front-end: React, Axios, React Router, HTML5/CSS3/JS
+ Back-end: Java, Spring Boot, BCrypt, Gradle
+ Database: MySQL
+ Libraries: JPA, JWT, SMTP(JavaMailSender), Selenium, Lombok
+ APIs: Kakao Maps API, YouTube Data API, 공공데이터 portal API
+ Tools: Git, GitHub, JIRA


###[주요 기능]
- - -
1. 내 주변 및 클릭한 지역 주변 콘텐츠, 지역화폐 가맹점 찾기 및 정주여건 확인
2. 직접 방문하여 인증샷을 찍어 인증하기
3. 인증한 내역을 관리자가 승인하여 포인트 지급
