# 하모예(HAMOYEAH)

### 👨‍🏫 프로젝트 소개
지역의 문화재, 착한식당, 제로 웨이스트 샵 등을 방문하여 인증하면 포인트를 지급합니다. 시민들의 참여를 통해 **지역 경제의 선순환 구조**를 만들고, 공공데이터를 활용해 내가 살고 있는 지역의 **정주여건을 시각화**하여 유용한 정보를 제공합니다.

---

### 💡 개발 목적
* **지역 경제 선순환 구조 구축** : 지역 화폐 사용 장려 및 소상공인 매출 증대 유도
* **데이터 기반 '정주여건' 제공** : CCTV, 가로등, 대기질, 소음 데이터를 융합하여 지역별 환경 등급 시각화
* **자발적 시민 참여형 구조** : 보상 시스템을 통해 능동적인 지역 서비스 이용 생태계 조

---

### 🧑‍💻 팀원/역할
| 이름 | 역할 | 담당 업무 |
| :--- | :---: | :--- |
| **장민서** | 👑 팀장 | 공공데이터 DB 적재, 프론트엔드 개발 및 백엔드 연동, JIRA 관리, YouTube API |
| **이연지** | 📣 발표 | 회원가입/로그인 시스템, 즐겨찾기, 관리자 인증 시스템 구축 |
| **심준현** | 🌿 GIT 담당 | 카카오맵 시각화(정주여건/마커), 위도·경도 거리 계산, 뉴스 크롤링 |

---

### 🛠️ 기술 스택
* **Front-end**: `React`, `Axios`, `React Router`, `HTML5/CSS3/JS`
* **Back-end**: `Java`, `Spring Boot`, `BCrypt`, `Gradle`
* **Database**: `MySQL`
* **Libraries**: `JPA`, `JWT`, `SMTP`, `Selenium`, `Lombok`
* **APIs**: `Kakao Maps API`, `YouTube Data API`, `공공데이터 Portal API`
* **Tools**: `Git`, `GitHub`, `JIRA`

---

### ⚙️ 주요 기능
1. **주변 탐색** : 내 주변 지역화폐 가맹점 및 정주여건 등급 확인
2. **방문 인증** : 직접 방문하여 인증샷 업로드 및 포인트 신청
3. **포인트 관리** : 관리자 승인을 통한 포인트 지급 및 내역 관리
